## Sprint 7: Contact Form & Inquiry Management

### 1. Kontext & Rolle
```
Als Full-Stack Developer unterstütze mich bei der Kontaktformular-Implementierung.

**Projektkontext:**
- Ziel: Kontaktformular mit Inquiry Management System
- Zielgruppe: Potenzielle Kunden & Administrator
- Umfang: Form, Email, Admin Management
- Deadline: 1 Woche (Sprint 7)
```

### 2. Aufgabengliederung
```
**Schritt 1 - Contact Form:**
1. **Form Implementation:**
   - Anforderung: Validierung mit Zod
   - Format: React Hook Form Integration
   - Kriterien: GDPR Checkbox, Categories

2. **Email Integration:**
   - SMTP/SendGrid Setup
   - Auto-Reply Templates
   - Admin Notifications

3. **Database Storage:**
   - Inquiries Table Implementation
   - Status Management
   - Priority Calculation

**Schritt 2 - Admin Inquiry Management:**
1. **Inquiry Dashboard:**
   - List View mit Filtern
   - Status Updates
   - Reply Interface

**Denke Schritt für Schritt durch den Inquiry Workflow von Eingang bis Antwort.**
```

### 3. Ausgabe-Spezifikation
```
**Deliverables:**
- Funktionierendes Kontaktformular
- Email-Versand (Auto-Reply)
- Admin Inquiry Management
- Status-Tracking

**Format:**
- Contact Page Components
- Email Templates
- Inquiry Management UI

**Qualitätskriterien:**
- Form Validation funktioniert
- Emails werden zugestellt
- GDPR-konform
```
